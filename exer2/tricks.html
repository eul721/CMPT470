Flexbox fun

Flexbox organizes child elements in a specific strict order. It is CSS3 specific and solves element collapsing problems. It allows developers to create organizable layouts without the use of frameworks.

Pseudo-Elements and Pseudo-Classes

Pseudo elements/classes enable developers to create decorations on elements without adding adjacent elements or extra classes.

  1. element 1: FIRST
  2. element 2: Do you like these carets?
  3. element 3: Notice how the odds have a light green?
  4. element 4: And evens don't?
  5. element 5: You like the colors on FIRST & LAST?
  6. element 6: LAST
